# Project Roadmap — Noor Oriental Rugs

This document outlines upcoming product catalog features and architectural plans designed to scale the website platform.

---

## Phase 1: Real-time Consultation Tools (Q3 2026)

* **Video Consultation Integration**: Add a calendar booking widget (e.g. Calendly API integration) to let clients schedule personal video calls to inspect rug details live from the Boston showroom.
* **Interactive Live Chat**: Embed a messaging widget (such as LiveChat, WhatsApp, or HubSpot) specifically on product pages to answer client inquiries instantly.

---

## Phase 2: Catalog Search & Filters (Q4 2026)

* **Refined Filtering Suite**: Add query controls on the primary listing catalog (`rugs-for-sale.html`) to sort inventory by style (Vintage, Traditional), size ranges (runner, small area, large room), and color shades.
* **Text Search Query Bar**: Enable live client-side fuzzy searching matching terms in names, origins, and descriptions.

---

## Phase 3: Showroom Reservation Systems (Q1 2027)

* **Dynamic In-Home Trial Selector**: Code a streamlined, step-by-step reservation checkout path that manages local 3-day home viewing requests.
* **Showroom Map Embed**: Render visual maps and directions alongside interactive tour indicators on the Visit Our Showroom section.

---

## Phase 4: Administrative Portal (CMS) (Q2 2027)

* **Headless CMS integration**: Move the static data in `inventory.json` to an API-based database manager (such as Strapi, Sanity.io, or Google Sheets API).
* **Inventory Editing Client**: Construct a passcode-protected administrative dashboard allowing rug store staff to upload photographs, write story details, set prices, and toggle item availability statuses without opening code files.
